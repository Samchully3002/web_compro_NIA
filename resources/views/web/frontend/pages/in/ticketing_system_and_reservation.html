Modernization of Indonesian Terminals
Modernization of Indonesian Terminals

We are carrying out an Indonesian terminal modernization project, the two Indonesian terminals that we have realized are the Kampung Rambutan terminal, East Jakarta and the Leuwipanjang terminal, Bandung. In this project we provide improvements to buildings, facilities and systems. One of the systems we provide is an online ticketing and reservation system.

  • Vending Machine
    A system that functions for printing tickets, ordering tickets independently using non-cash payment methods.
  • Website
    A system that functions for ordering tickets online which is supported by an online payment system, apart from that it also functions for ticket cancellation. Smart Terminal
  • Counter Management System
    Sales System for Tickets from counter agents which functions for ordering tickets, issuing tickets, canceling tickets and reprinting tickets. At this counter the payment system can use cash or non-cash methods.
  • Driver Application
    This application is used for bus conductors or drivers to check boarding tickets which can be scanned via QR/Barcode. Through this driver application, the driver can find out the number of passengers according to the number of tickets that have been ordered in real time.
